Markdown 转小红书文字卡片

从原始想法到可发布卡片

这些示例展示了开源本地版如何把结构化 Markdown 渲染为适合小红书发布的文字卡片。官方在线版可以进一步把无格式原文自动整理成类似结构。

示例 1：护肤步骤整理
生活干货
Markdown 输入

以前一直乱抹，后来发现：**顺序真的很重要**。

## 正确步骤

- 水 → 乳 → 乳液 → 面霜
- 去角质：一周 1 次
