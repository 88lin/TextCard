Markdown 转小红书文字卡片

从原始想法到可发布卡片

这些示例展示了如何把结构化 Markdown 渲染为适合小红书发布的文字卡片。你可以照着示例写内容，也可以把 AI 排版生成的 Markdown 放进编辑器里继续微调。

示例 1：护肤步骤整理
生活干货
Markdown 输入
# 护肤顺序错了？一个月变化超明显

以前一直乱抹，后来发现：**顺序真的很重要**。

## 正确步骤

- 水 → 乳 → 乳液 → 面霜
- 去角质：一周 1 次

==坚持一个月，皮肤状态明显提升==
