TextCard Studio 使用指南

TextCard Studio 不需要登录、不上传内容。你只需要输入 Markdown，选择模板，浏览器会在本地 Canvas 中渲染并导出 1242×1656 高清图片。

如果不确定应该怎么写 Markdown，可以先看 排版示例，再回到编辑器开始生成图片。AI 辅助排版适合输出结构化 Markdown，再交给编辑器预览和导出。

本地渲染

所有排版和导出都在浏览器完成，适合处理私密草稿。

Markdown 排版

支持标题、加粗、高亮、引用、列表、图片和强制分页。

模板定制

可调整字号、行高、颜色、内边距、签名和水印。

1. 快速开始

  1. 打开 editor.html。
  2. 在左侧输入或粘贴 Markdown 文本。
  3. 在右侧选择模板，并按需调整字号、颜色、行高、签名和水印。
  4. 确认预览无误后，点击「打包下载全部图片」。

2. 常用 Markdown 语法

# 一级标题
## 二级标题

这是一段正文，支持 **加粗**、==高亮==、*斜体*、~~删除线~~ 和 `行内代码`。

> 这是一段引用，适合金句、摘录和重点提示。

- 无序列表
- 继续列出观点

1. 有序步骤
2. 第二个步骤

---

上面的三个短横线会强制开启新页面。
分页建议：长文尽量用空行分隔段落。系统会优先在段落之间分页，阅读节奏会比连续大段文字更自然。

3. 模板选择建议

极简杂志

适合观点文、知识总结、轻商业内容。

书籍内页

适合摘录、读书笔记、深度长文。

大厂文档

适合职场方法论、商业分析、教程。

暗夜深思

适合科技、开发者、深夜思考类内容。

复古拍立得

适合旅行、生活记录、情绪化表达。

空白模板

适合从零开始做个人品牌统一视觉。

4. 数据与导出说明

内容处理：编辑、预览和导出都在当前浏览器中完成，适合处理还没有发布的草稿。

图片导出：默认输出 1242×1656 高清图片，可按当前选择的 PNG 或 JPEG 格式下载。

AI 扩展：AI 排版适合放在输入前置环节，把原始文案整理成标题、段落、重点和列表；最终图片仍由编辑器统一渲染。
